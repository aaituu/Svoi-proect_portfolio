# Портфолио — ТОО «СВОЙ ПРОЕКТ»

Сайт-портфолио проектной компании **ТОО «СВОЙ ПРОЕКТ»** (Астана) — проектирование
жилых, коммерческих и промышленных объектов под ключ, генеральные планы,
благоустройство и наружные инженерные сети.

Это **адаптивный сайт** (телефон / планшет / компьютер), у которого есть
**готовый к скачиванию буклет** — горизонтальный PDF формата A4 (альбомная
ориентация), 10 страниц.

## Скачать буклет

Кнопки **«Скачать буклет портфолио (PDF)»** (внизу страницы и в верхней панели)
отдают **готовый файл [`booklet.pdf`](booklet.pdf)** — уже сверстанный буклет с
правильными размерами A4-landscape. Ничего настраивать в браузере не нужно, сайт
заново не «печатается» — скачивается именно готовый PDF.

### Как пересобрать booklet.pdf (если менялся контент)

Буклет генерируется из `index.html` через Chrome (headless) — печать использует
`@media print` с `@page { size: A4 landscape }`, поэтому кнопки и панель в PDF не
попадают:

```sh
"/Applications/Google Chrome.app/Contents/MacOS/Google Chrome" \
  --headless --disable-gpu --no-pdf-header-footer \
  --run-all-compositor-stages-before-draw --virtual-time-budget=15000 \
  --print-to-pdf="booklet.pdf" "file://$PWD/index.html"
```

Получается ровно 10 страниц A4 (альбомная).

## Содержимое

- `index.html` — весь сайт (один файл, без сборки)
- `assets/projects/` — изображения объектов
- `assets/renders/` — генпланы и инженерные сети (из проектной документации)
- `assets/licenses/` — сканы лицензии и приложений
