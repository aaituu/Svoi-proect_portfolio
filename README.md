# Портфолио — ТОО «СВОЙ ПРОЕКТ»

Сайт-портфолио проектной компании **ТОО «СВОЙ ПРОЕКТ»** (Астана) — проектирование
жилых, коммерческих и промышленных объектов под ключ, генеральные планы,
благоустройство и наружные инженерные сети.

Это **адаптивный сайт** (корректно открывается на телефонах, планшетах и
компьютере), который при скачивании превращается в **горизонтальный буклет**
формата A4 (альбомная ориентация) — стиль презентации-каталога.

- **На компьютере и планшете** — каждая страница показана как альбомный слайд
  16:9; содержимое масштабируется пропорционально (container-query units), на
  экране выглядит ровно так же, как при печати.
- **На телефоне** — слайды перестраиваются в удобную вертикальную ленту с
  крупным читаемым текстом (адаптивная вёрстка).
- **При скачивании / печати** — `@page { size: A4 landscape }`: получается
  горизонтальный PDF-буклет, по одному слайду на лист.

## Печать / PDF-буклет

Откройте сайт и нажмите кнопку **«Скачать буклет (PDF)»** в верхней панели
(или `Ctrl/Cmd + P`). В диалоге печати:

- Ориентация — **Альбомная / Landscape** (выбирается автоматически)
- Размер бумаги — **A4**
- Поля — **Нет / None**
- Включите **«Фоновая графика» / «Background graphics»** (чтобы напечатались
  цвета, обложка и фотографии)
- Для PDF выберите «Сохранить как PDF»

Получится готовый горизонтальный буклет на 10 страниц.

## Содержимое

- `index.html` — весь сайт (один файл, без сборки)
- `assets/projects/` — изображения объектов
- `assets/renders/` — генпланы и инженерные сети (из проектной документации)
- `assets/licenses/` — сканы лицензии и приложений
- `assets/logo.png`, `assets/favicon.svg` — фирменный стиль
