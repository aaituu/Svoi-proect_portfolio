# Портфолио — ТОО «СВОЙ ПРОЕКТ»

Сайт-портфолио проектной компании **ТОО «СВОЙ ПРОЕКТ»** (Астана) — проектирование
жилых, коммерческих и промышленных объектов под ключ, генеральные планы,
благоустройство и наружные инженерные сети.

Страница оформлена как **буклет формата A4**: на экране это аккуратная стопка
страниц, а при печати каждая страница ложится ровно на один лист A4.

## Печать / PDF-буклет

Откройте сайт и нажмите кнопку **«Печать / Скачать буклет»** в верхней панели
(или `Ctrl/Cmd + P`). В диалоге печати:

- Размер бумаги — **A4**
- Поля — **Нет / None**
- Включите **«Фоновая графика» / «Background graphics»** (чтобы напечатались
  цвета и обложка)
- Для PDF выберите «Сохранить как PDF»

Получится готовый буклет на 10 страниц.

## Структура буклета

1. Обложка
2. О компании · преимущества
3. Что мы проектируем · состав работ
4. Как мы работаем (этапы)
5. Избранный проект — генплан, благоустройство и инженерные сети
6–8. Реализованные проекты (жильё, коммерция, промышленность, сети)
9. Лицензии и допуски
10. Контакты

## Содержимое

- `index.html` — весь сайт (один файл, без сборки)
- `assets/projects/` — изображения объектов
- `assets/renders/` — генпланы и инженерные сети (из проектной документации)
- `assets/licenses/` — сканы лицензии и приложений
- `assets/logo.png`, `assets/favicon.svg` — фирменный стиль
